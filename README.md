scss support 🐠

## Getting Started

1️⃣First, install npm:

```bash
npm install
```

2️⃣run the development server:

```bash
npm run dev
```

3️⃣expoer static html:
```bash
npm run export
```

4️⃣run static html server:
```bash
npm run serve
```
